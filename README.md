# Bleachers — bleachers.shop

Sitio web de marca para Bleachers (streetwear · 100% hand dyed).

## Ver en local

```bash
cd site
python3 -m http.server 8080
```

Abrí http://localhost:8080 en el navegador.

## Publicar

### GitHub Pages

1. Este repo debe tener `index.html` en la raíz (carpeta `site/`).
2. En GitHub: **Settings → Pages → Build and deployment → Source: Deploy from a branch**.
3. Branch: `main`, folder: `/ (root)`.
4. En tu DNS, apuntá **bleachers.shop** al dominio que te da Pages (o configurá custom domain en Pages).

### Otro hosting

Subí el contenido de esta carpeta a Vercel, Netlify, Cloudflare Pages, etc. y apuntá el dominio **bleachers.shop** al deploy.
